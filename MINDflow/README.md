## Installazione e Avvio

### Prerequisiti
- Node.js (versione 16 o superiore)
- npm o yarn
- Expo CLI (installato globalmente)

### Setup
1. Clona il repository
2. Installa le dipendenze:
   ```bash
   npm install
   ```

3. Avvia l'applicazione:
   ```bash
   # Per Android
   npm run android
   
   # Per iOS (richiede macOS)
   npm run ios
   
   # Per web
   npm run web
   ```
